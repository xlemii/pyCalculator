# Simple Calculator Python

A simple, colorful calculator built in Python using the **Tkinter** library.
It includes click sounds, an animated result, and a confetti effect after a successful calculation.

## Features

* Basic mathematical operations:

  * addition
  * subtraction
  * multiplication
  * division
* Enter numbers using the keyboard or on-screen buttons
* Animated result text
* Confetti effect after a successful calculation
* Click sound on every button press
* Keyboard shortcuts:

  * **Enter** – calculate result
  * **Esc** – clear the input

## Requirements

* Python 3.x
* Windows system (due to the `winsound` library)

## Running the program

1. Clone the repository:

```bash
git clone https://github.com/your-username/super-calculator.git
```

2. Go to the project folder:

```bash
cd super-calculator
```

3. Run the program:

```bash
python calculator.py
```
